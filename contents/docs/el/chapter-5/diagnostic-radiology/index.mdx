Η διαγνωστική ακτινολογία και οι καθοδηγούμενες με απεικόνιση επεμβατικές πράξεις χρησιμοποιούν ακτίνες Χ για απεικόνιση και, σε ορισμένες περιπτώσεις, καθοδήγηση θεραπείας. Από την ανακάλυψή τους από τον Wilhelm Conrad Röntgen το 1895, οι ακτίνες Χ έχουν καταστεί απαραίτητες στην ιατρική πρακτική λόγω της ικανότητάς τους να διεισδύουν στους ιστούς και να παράγουν εικόνες εσωτερικών δομών.

Οι ακτινογραφικές τεχνικές παρέχουν στατικές εικόνες του σώματος, ενώ η ακτινοσκόπηση επιτρέπει απεικόνιση σε πραγματικό χρόνο, υποστηρίζοντας τόσο τη διάγνωση όσο και ελάχιστα επεμβατικές διαδικασίες. Κοινές διαδικασίες ακτινολογίας περιλαμβάνουν τη γενική ακτινογραφία, την αξονική τομογραφία (CT), τη μαστογραφία και επεμβατικές διαδικασίες όπως αγγειογραφία, τοποθέτηση stent και βιοψία καθοδηγούμενη με απεικόνιση (IAEA, 2018).

## Παραγωγή Ακτίνων Χ

Οι ιατρικές εικόνες σχηματίζονται βάσει των διαφορετικών ιδιοτήτων εξασθένισης των ιστών, που εξαρτώνται από την πυκνότητα και την ατομική σύστασή τους.

Οι ακτίνες Χ παράγονται σε μια λυχνία ακτίνων Χ, όπου ηλεκτρόνια που εκπέμπονται από θερμαινόμενη κάθοδο επιταχύνονται προς μεταλλική άνοδο. Όταν αυτά τα ηλεκτρόνια υψηλής ενέργειας αλληλεπιδρούν με τον στόχο, παράγονται ακτίνες Χ που εξέρχονται μέσω ενός παραθύρου για να σχηματίσουν την πρωτογενή δέσμη.

Καθώς η δέσμη ακτίνων Χ διέρχεται μέσα από τον ασθενή, ορισμένα φωτόνια απορροφώνται, ορισμένα σκεδάζονται και ορισμένα διέρχονται για να σχηματίσουν την εικόνα. Η σκεδαζόμενη ακτινοβολία εκπέμπεται σε διαφορετικές κατευθύνσεις, καθιστώντας τον ασθενή την κύρια πηγή δευτερογενούς ακτινοβολίας στον χώρο. Αν και η σκεδαζόμενη ακτινοβολία είναι πολύ λιγότερο ισχυρή από την πρωτογενή δέσμη, αποτελεί την κύρια πηγή επαγγελματικής έκθεσης, δεδομένου ότι το υγειονομικό προσωπικό σπάνια εκτίθεται άμεσα στην πρωτογενή δέσμη.

Εντός της λυχνίας ακτίνων Χ, η ακτινοβολία εκπέμπεται προς όλες τις κατευθύνσεις· για τον λόγο αυτό, η λυχνία είναι εξωτερικά θωρακισμένη, εκτός από το παράθυρο, ώστε να αποτρέπεται η διάδοση ακτινοβολίας σε ανεπιθύμητες κατευθύνσεις. Ωστόσο, ένα μικρό ποσοστό φωτονίων υψηλής ενέργειας μπορεί να διεισδύσει στο προστατευτικό κέλυφος, με αποτέλεσμα τη λεγόμενη ακτινοβολία διαρροής (Nowotny, 2014· Poletti, 2014).

## Μέτρα Ακτινοπροστασίας στην Ακτινογραφία

Οι ακτινογραφικές διαδικασίες εκτελούνται τυπικά σε ειδικές αίθουσες ακτίνων Χ, όπου η δομική θωράκιση παρέχει επαρκή προστασία. Η πρόσβαση σε αυτές τις αίθουσες είναι περιορισμένη και προειδοποιητικές σημάνσεις υποδεικνύουν την παρουσία ιονίζουσας ακτινοβολίας.

Σε ορισμένες περιπτώσεις, υγειονομικό προσωπικό μπορεί να χρειαστεί να βοηθήσει ασθενείς (π.χ. ηλικιωμένους ασθενείς ή παιδιά). Σε αυτές τις περιπτώσεις, πρέπει να τηρούνται οι ακόλουθες προφυλάξεις:

→ Χρήση κατάλληλου προστατευτικού εξοπλισμού (π.χ. μολύβδινη ποδιά)

→ Διατήρηση της μέγιστης δυνατής απόστασης από τη δέσμη ακτίνων Χ

→ Τα χέρια να παραμένουν εκτός της πρωτογενούς δέσμης

→ Αποφυγή επανειλημμένης έκθεσης του ίδιου μέλους του προσωπικού

→ Η εγκυμοσύνη ή η υποψία εγκυμοσύνης πρέπει να αναφέρεται στο ακτινολογικό τμήμα

→ Τήρηση των οδηγιών του αρμόδιου προσωπικού ακτινοπροστασίας (IAEA, 2018· ICRP, 2013)

### Κινητές Μονάδες Ακτίνων Χ

Οι κινητές μονάδες ακτίνων Χ χρησιμοποιούνται συχνά σε χώρους όπως μονάδες εντατικής θεραπείας, τμήματα επειγόντων περιστατικών ή θαλάμους νοσηλείας, όπου η δομική θωράκιση μπορεί να είναι περιορισμένη. Σε αυτές τις περιπτώσεις, πρέπει να τηρούνται οι ακόλουθες προφυλάξεις:

→ Το προσωπικό που δεν απαιτείται κοντά στον ασθενή πρέπει να διατηρεί τη μέγιστη δυνατή απόσταση

→ Όσοι πρέπει να παραμείνουν πρέπει να φορούν κατάλληλο προστατευτικό εξοπλισμό

→ Χρήση κινητής θωράκισης όταν είναι διαθέσιμη

→ Τήρηση των οδηγιών του αρμόδιου προσωπικού ακτινοπροστασίας (IAEA, 2018)

## Μέτρα Ακτινοπροστασίας στην Ακτινοσκόπηση

Η ακτινοσκόπηση και οι καθοδηγούμενες με απεικόνιση επεμβατικές πράξεις εκτελούνται σε ελεγχόμενες περιοχές με περιορισμένη πρόσβαση, ενώ κινητά ακτινοσκοπικά συστήματα (C-arm) χρησιμοποιούνται σε χειρουργεία και χώρους επειγόντων.

<img alt='Σχήμα 29: Ακτινοσκοπικό σύστημα C-arm' src='/figures_el/29.png' />
*Σχήμα 29: C-arm. Από Mobile X-ray image intensifier, Dennisthemonkeychild (2015), Wikimedia Commons. CC BY-SA 4.0.*

Δεδομένου ότι η έκθεση του προσωπικού οφείλεται κυρίως σε σκεδαζόμενη ακτινοβολία από τον ασθενή, τα ακόλουθα μέτρα είναι απαραίτητα:

→ Παραμονή στον χώρο μόνο όταν είναι απαραίτητο

→ Μεγιστοποίηση της απόστασης από τον ασθενή κατά τη διάρκεια της έκθεσης

→ Χρήση προστατευτικών μολύβδινων ποδιών

→ Χρήση πρόσθετου προστατευτικού εξοπλισμού, όπως θυρεοειδικές ασπίδες, προστατευτικά γυαλιά και γάντια, όταν ενδείκνυται

→ Κανένα μέρος του σώματος δεν πρέπει να τοποθετείται στην πρωτογενή δέσμη

→ Το προσωπικό που είναι ή ενδέχεται να είναι έγκυο πρέπει να ενημερώνει τον αρμόδιο επόπτη

→ Τήρηση των οδηγιών του αρμόδιου προσωπικού ακτινοπροστασίας (IAEA, 2018· ICRP, 2013)
